入境规定

二十岁以上人员的入境免税品规定： 1. 二百支香烟，或者二十五支雪茄，或者一磅烟草； 2. 一瓶（不超过1公升）的饮用酒； 3. 合理数量的香水； 4. 非商用目的的入境货物，其价值不超过两万新台币（TWD）（二十岁以下者限额为一万新台币）
二十岁以上人员的入境免税品规定：
1. 二百支香烟，或者二十五支雪茄，或者一磅烟草；
2. 一瓶（不超过1公升）的饮用酒；
3. 合理数量的香水；
4. 非商用目的的入境货物，其价值不超过两万新台币（TWD）（二十岁以下者限额为一万新台币），条件是距乘客最后一次入境台湾三十天以上。

禁止入境1. 毒品以及赌博物品和玩具手枪；
2. 阿尔巴尼亚、亚美尼亚、阿塞拜疆、白俄罗斯、保加利亚、柬埔寨、中华人民共和国、古巴、哈萨克斯坦、格鲁吉亚、朝鲜民主主义人民共和国 、吉尔吉斯斯坦，老挝人民民主共和国、摩尔多瓦、罗马尼亚、俄罗斯联邦、塔吉克斯坦、土库曼斯坦、乌克兰、乌兹别克斯坦以及越南的任何产品；
3. 除猫和狗之外的活的动物；
4. 除干燥的以及水产加工产品之外的动物产品；
5. 活的植物、新鲜植物产品以及新鲜水果。
对于金、银、枪支和弹药有特殊规定。要求乘客填写书面的行李申报，包括另送行李。
 

出境规定

出境免税黄金饰品最高为六十二点五克，银饰品最高位六百二十五克。 禁止出境： 金银条、金银硬币、未冲洗胶卷；枪支；爆炸物以及军用物品。 乘客不遵守允许的金银携带数量将导致没收以及依据相关法律的处罚。
