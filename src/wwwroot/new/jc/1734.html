  8. 机场地址：  Mingaladon Township, 缅甸

仰光国际机场交通

公交：从机场出来向左转(国内航站楼的方向)步行5分钟左右，会看见很多皮卡车，乘坐到“Sel Mile Gone”公车站，费用为100K，然后转乘5 1路车( 1 5 0 K - 3 0 0 K )可以到市区的中国城( C h i n aTown)，不过51路车并没有阿拉伯数字标示，是用的缅语写的，可以参照语言帮助部分的缅语和阿拉伯数字对照图。除了51路，还有很多其他的公交也可以到市内，听见工作人员叫“Sule”的，都可以上车，到了苏雷塔，再去往其他地方就很方便了。

出租车：机场到市区到价格为5000K-6000K不等，不打表，直接讲价，司机要价都在7000K-9000K左右，记得砍价，还可考虑和别人拼车。

机场接送：一般预订的酒店都有免费出租和巴士接送，这个需要你在预订酒店的时候就提前确认。

仰光国际机场评价

仰光国际机场怎么样？飞瀛网收集提供仰光国际机场用户评价如下：

@柏舟：仰光机场可以算是麻雀虽小，五脏俱全吧。办理登机的柜台依旧手写登机牌。大厅和候机厅都有柜台可以把剩下的缅币换回美元。差不多一半以上的店铺都在卖玉石和宝石制品，至于品质嘛，就见仁见智啦。

@凝花：仰光机场国际航站楼还行，国内航站楼就像一个汽车站，摆渡车是从日本淘汰来的70年代的公交车，登机靠吼。晚上11点以后到达机场,基本所有的店铺都关门了，只有剩下咖啡馆还开着。

@明阳：仰光机场里的壁画很地方特色哦，虽然地方较小，但是很整洁，机场里的工作人员挺友好的，还会说中文，就是设施落后了一点，WIFI很慢，免税店很小，出了门就有出租车了，去市区也很近，很快就到达了。

@老渔：说这个机场是用垃圾填海建造的哦！好高科技！这个机场蛮有日本特色的，机场大厅好多忍者什么的，很日本。值得一提的是日本人服务态度真心好。因为赶早班机，所以就住在机场附近。早上起来看了日出，结束了我的日本支...

@金鱼：缅甸最好的机场。干净，快捷。国际厅很好，国内厅一般。有wifi。有人说入关要给小费，我没给也没受到刁难。酒店出来沿右边一直走，有个「十里香」饭馆，做中餐，味道很不错，而且有wifi。

@西惜夕兮：机场就没什么特色的，最令我有好感的是机场的换汇率居然比市区的还要好，而且买卖的差价是我去过的机场中最小的，没用完的缅币换回美元也没这么亏！

@小水晶：从Heho飞回仰光的时候我还跟同伴抱怨，这机场要多破有多破，还怎么国际啊？等到飞曼谷去International那边的时候，人家是全新的啊，brandnew啊，还有freewifi，所以挣了一颗星...
